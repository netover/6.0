# Checking Status

## Quick Start
1. Obtenha status de workstations (`get_workstation_status`).
2. Se job específico: extraia log/status (`get_job_log` ou equivalente).
3. Responda com: estado atual, evidências, próximos passos.

## Perguntas de clarificação
- Se não houver job/workstation informado, peça o identificador.
